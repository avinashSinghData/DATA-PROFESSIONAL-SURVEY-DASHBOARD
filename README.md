# DATA-PROFESSIONAL-SURVEY-DASHBOARD
Interactive dashboard analyzing a data professional survey with customized design and visual layout.

**PROBLEM STATEMENT**
Survey responses from data professionals contain valuable insights into roles, salaries, tools, and job satisfaction, but the data is difficult to interpret in raw form. This project presents an interactive dashboard that organizes survey data into clear visual summaries, enabling easy comparison across roles and experience levels.

**APPROACH**
Prepared and structured survey data for analysis.
Defined key metrics and categorical breakdowns relevant to data roles.
Designed a customized dashboard layout with improved color schemes and visual positioning.
Validated insights across different segments of the survey.

**KEY-INSIGHTS**
Data Scientists and Data Engineers show higher average compensation.
Python is the most widely used language across data roles.
Nearly half the respondents are happy with work life balance and salary.
Survey takers are mostly from countries like US, UK and India and Canada.
Almost half of the respondents suggest that it is neither easy not difficult to break into data.

**KEY-LEARNINGS**
Working with survey-based, non-transactional datasets.
Designing dashboards focused on categorical distributions rather than KPIs.
Improving dashboard readability through layout and color customization.
Translating survey responses into concise visual insights.
